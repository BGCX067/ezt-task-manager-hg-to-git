Things user can do:

Set of default shortcut keys:

Ctrl d C Show or hide Task Manager
Ctrl q C focus on the Day display panel
Ctrl w C focus on the Week display panel
Ctrl e C focus on the Year display panel
Ctrl y C focus on the Search display panel
Ctrl t C focus on the Event display panel
Ctrl a C focus on the command box
Ctrl r C focus on the Calendar Date (after that press a few time of any arrow key)
Page Up C focus on the calendar prev month button then press enter
Page Down C focus on the calendar next month button then press enter
Home C focus on the calendar year combo box then press enter
SPACEBAR C to active the ACTION POP UP MENU which consists of the update, delete and status 
                       buttons
Esc key C close the ACTION POP UP MENU
Shift + backspace = backspace

 

1.       Create Task

Type ‘add’ in the command box -> hit enter -> key in task description & hit enter-> from date (dd-mmm-yy) hit enter-> end date (dd-mmm-yy) hit enter -> time (hh-hh) hit enter -> priority (high/medium/low) hit enter -> alert (true/false) -> hit enter. - > Successful message prompt and will updated the task panel.

                          

2.       Update Task

Left click or SPACEBAR on the task - > an action box will prompt -> select update button -> the update command will auto generate in the command box -> change the task details which you want to update - > hit enter.

3.       Delete Task

Left click or SPACEBAR on the task - > a action box will prompt -> select delete button -> the delete command will auto generate in the command box -> hit enter.

 

4.       Update Task Status

Left click or SPACEBAR on the task - > a action box will prompt -> select status button -> the update status command will auto generate in the command box -> change the task status to active or non active - > hit enter.

 

5.       Create Event

Type add in the command box -> description -> from date (dd-mmm-yy) -> end date (dd-mmm-yy) -> time (nil) -> priority (high/medium/low) -> alert (true/false) -> hit enter. - > Successful message prompt and will updated the task panel.

Note: the difference between Task and Event is the time of an event is nil.

6.       Update Event

Left click or SPACEBAR on the event from the event panel - > a action box will prompt -> select update button -> the update command will auto generate in the command box -> change the task details which you want to update - > hit enter.

7.       Delete Event

Left click or SPACEBAR on the event - > a action box will prompt -> select delete button -> the delete command will auto generate in the command box -> hit enter.

8.       Update Event Status

Left click or SPACEBAR on the event - > a action box will prompt -> select status button -> the update status command will auto generate in the command box -> change the task status to active or non active - > hit enter.

10.   Set Handphone number for reminder SMS

Type ‘h no which you want to change’ in command box. E.g. h 80809900

 

11.   Set Email address for reminder Email

Type ‘e email which you want to change’ in command box.
               E.g. e peter@hotmail.com
